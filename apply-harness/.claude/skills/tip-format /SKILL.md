# tip-format

Claude Code 팁 문서의 표준 포맷.

## 파일 위치/이름
- 경로: `tips/<kebab-case>.md`
- 이름: 영문 소문자 + 하이픈 (예: `hooks-pre-tool-use.md`)

## 프론트매터 (필수)
```yaml
---
title: <한글 제목>
tags: [훅, 설정]   # 카테고리 태그 1~3개
updated: YYYY-MM-DD
source: <출처 URL 또는 경로>
---
```

## 본문 구조
```markdown
# <제목>

## 상황
어떤 문제/니즈에서 이 팁이 필요한가.

## 방법
단계별로 무엇을 한다.

## 예시
\`\`\`bash
# 실제 명령
\`\`\`
\`\`\`json
// 실제 설정
\`\`\`

## 주의점
함정, 버전 의존성, 검증이 필요한 부분.
```

## 규칙
- 한국어, 간결한 설명체.
- 코드블록 언어 명시.
- 한 파일 = 한 팁.
- 추측/미검증은 "확인 필요:" 프리픽스로 표시하거나 제외.
