# 이미지 붙여넣기로 스크린샷 기반 작업

## 상황
UI 깨짐, 에러 화면, 디자인 시안처럼 말로 설명하기 어려운 것은 이미지로 보여주는 게 빠르다. Claude Code는 이미지를 입력으로 받아 분석한다.

## 방법
- 클립보드의 이미지를 입력창에 **붙여넣기**(macOS: `Cmd + V`)한다.
- 또는 이미지 파일을 터미널/창에 **드래그 앤 드롭**한다.
- 이미지 파일 경로를 `@경로`로 참조할 수도 있다.

## 예시
```text
# 스크린샷을 찍어 클립보드에 복사한 뒤
Cmd + V          → 입력창에 이미지 첨부
> 이 화면에서 버튼 정렬이 깨졌어. 원인 찾아줘

# 파일 경로로 참조
> @./design/mockup.png 이 시안대로 헤더를 구현해줘
```

## 주의점
- 붙여넣기/드래그 지원은 터미널 에뮬레이터·IDE 확장에 따라 다르다. 일반 터미널에서 안 되면 파일 경로(`@`) 참조를 쓴다.
- 민감정보(토큰, 개인정보)가 담긴 스크린샷은 가리고 올린다.
- 확인 필요: 지원 이미지 포맷·최대 크기는 환경에 따라 다를 수 있다.
