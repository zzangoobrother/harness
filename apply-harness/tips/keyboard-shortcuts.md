# 자주 쓰는 단축키와 입력 팁

## 상황
마우스 없이 키보드만으로 중단·수정·모드 전환을 빠르게 하면 작업 흐름이 끊기지 않는다.

## 방법
대표 단축키(macOS 기준):

- `Esc` — 현재 생성/실행을 즉시 중단한다.
- `Esc` 두 번 — 이전 메시지로 돌아가 편집한다(대화 분기).
- `Shift + Tab` — 권한/플랜 모드를 순환 전환한다. → [[plan-mode]]
- `Ctrl + C` — 입력 취소/세션 종료.
- 작업 도중 입력하면 **큐잉**되어 현재 작업이 끝난 뒤 처리된다.

멀티라인 입력:
- 줄바꿈은 `\` + Enter 또는 `Option + Enter` 등으로 넣는다(터미널/설정에 따라 다름). 확인 필요: 정확한 조합은 환경별로 다르니 `/help`로 확인한다.

## 예시
```text
# 엉뚱한 방향으로 작업이 진행될 때
Esc            → 즉시 멈춤

# 직전 내 메시지를 고쳐 다시 보내기
Esc Esc        → 이전 메시지 편집

# 실행 전 계획부터 받기
Shift + Tab    → 플랜 모드로 전환
```

## 주의점
- 단축키는 터미널 에뮬레이터·IDE 확장·OS에 따라 다를 수 있다. 현재 환경 기준은 `/help`에서 확인한다.
- 키바인딩은 `~/.claude/keybindings.json`으로 커스터마이즈할 수 있다(확인 필요: 지원 범위는 버전에 따라 다름).
